<Update label="2026-04-20" tags={["Feature"]} description="Week of April 20">
  ## Scheduled payouts

  You can now queue one-time USDC transfers for a specific date and time. Create a scheduled payout by providing a recipient, amount, and target datetime — Archer handles execution automatically when the time arrives. Cancel any pending payout before it runs if plans change.

  ## On-chain paying limits

  Set per-transaction, per-period, and call-count spending limits that are enforced by the ArcherAgentEndpoint smart contract. These controls are written on-chain, so no payout can exceed your authorized budget regardless of how it's triggered. [Learn more](/concepts/paying-limits)

  ## Bulk team member import via CSV

  Add multiple payout recipients at once using the CSV import feature on the Team Members page. Imported members are created with active status immediately — no separate activation step required. [Learn more](/guides/invite-team-members)

  ## ACH and wire off-ramp

  Move funds from your smart wallet back to a traditional bank account. The dashboard now supports both ACH and wire transfers for off-ramping USDC to fiat. [Learn more](/guides/fund-and-offramp)

  ## REST API

  A full REST API is available for managing accounts, payouts, team members, and wallet operations programmatically. All endpoints accept and return JSON, with JWT-based authentication.
</Update>
